# QA Copilot n8n

Workflow do n8n para automação de QA usando múltiplos agentes.

## Como importar no n8n

1. Abra o n8n.
2. Crie um novo workflow.
3. Clique nos três pontinhos do workflow.
4. Selecione Import from file.
5. Escolha o arquivo `workflows/qa-copilot.json`.
6. Configure as credenciais necessárias.

## Credenciais necessárias

- OpenAI
- Google Sheets

## Observação

As credenciais não devem ser enviadas para o GitHub.
